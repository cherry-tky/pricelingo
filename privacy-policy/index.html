PriceLingo

Privacy Policy

Effective June 11, 2026

PriceLingo is a browser extension that converts a price selected by the user into a preferred currency.

Data processed locally

Selected text is processed locally in your browser to determine whether it contains a price and currency. PriceLingo does not transmit the complete selected text or webpage URL.

Chrome synchronized extension storage saves your preferred target currency, enabled setting, and interpretations for ambiguous currency symbols. These preferences are used only to provide the extension's functionality.

Data sent for conversion

When you request a conversion, PriceLingo sends only the source and target three-letter currency codes, such as USD and EUR, to the Frankfurter API at api.frankfurter.dev. The selected amount is multiplied by the returned rate locally.

Frankfurter states that its API does not collect personal data. Its public service runs behind Cloudflare, which may receive basic request metadata for service delivery, security, and analytics.

Collection and sharing

  • No analytics, browsing-history collection, ads, or user accounts
  • No sale of user data
  • No remotely hosted code
  • No sharing except the disclosed currency-pair request

Chrome Web Store Limited Use

PriceLingo's use of information received from Chrome APIs complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. Website content is processed only to provide the user-requested price-conversion feature. It is not used or transferred for advertising, profiling, creditworthiness, or unrelated purposes, and humans cannot access it through PriceLingo.

Permissions

storage saves preferences. Website access lets PriceLingo recognize user-selected prices and display the conversion button. Access to api.frankfurter.dev obtains exchange rates.

Data retention

Preferences remain until changed or the extension is removed. Exchange rates are held temporarily in memory for up to six hours.

Contact

For support or privacy questions, visit the PriceLingo public support page.

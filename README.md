- Sitemap: https://pricelingo.com/sitemap.xml

## Search discovery

After the Chrome Web Store listing is public:

1. Confirm the listing links back to `https://pricelingo.com/`.
2. Verify `pricelingo.com` in Google Search Console.
3. Submit `https://pricelingo.com/sitemap.xml`.
4. Use URL Inspection to request indexing for the homepage.

Chrome Web Store indexes a newly public listing automatically, which may take a few hours.
